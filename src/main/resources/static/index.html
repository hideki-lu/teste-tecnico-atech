Esta solução exibe a quantidade de passageiros que estiveram ocupando a sala de espera durante o mesmo período de tempo.

Para fins de simplificar a implementar da solução foram consideradas apenas horas de entrada e saída dos passageiros, ou seja, não serão aceitas por enquanto valores de minutos e segundos referntes aos horarios.

Portanto, esta solução irá considerar os intervalos de tempo especificados nos campos de texto de entrada e saída.

A orientação para inserir as informações corretamente é que se insita primeiramente todas as horas em que cada passageiro na sala de espera, e no campo de texto abaixo de saídas sejam inseridas todas as horas de saída de cada passageiro da saída.

Futuramente alterações podem ser feitas para melhores adaptações ao formato de inserir os dados dos passeiros, esta é uma prova de conceito para esta solução.
